# Генератор случайных вызовов

## Обзор проекта

Небольшое веб-приложение для генерации случайных квестов по саморазвитию. Пользователь может выбрать сложность и категории, получить набор заданий, отмечать их выполнение и сохранять прогресс в браузере.

## Технологии

- `HTML`
- `CSS`
- `JavaScript`
- `localStorage`

## Как запустить

1. Скачайте или откройте проект на компьютере.
2. Откройте файл `index.html` в браузере.

При необходимости можно запускать проект через расширение `Live Server`, но для работы приложения это не обязательно.
